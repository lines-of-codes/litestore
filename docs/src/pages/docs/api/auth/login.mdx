</h1>

## Body

This route requires a JSON body with the following parameters.

| Name     | Type     | Required |
| -------- | -------- | -------- |
| username | `string` | Yes      |
| password | `string` | Yes      |

## Response

This route can return HTTP code 200 (OK), 400 (Bad Request),
401 (Unauthorized),
405 (Method Not Allowed), 500 (Internal Server Error)
