<h1>
	<Badge variant="get">GET</Badge> /api/files/list/\*
</h1>

Lists a file in a directory.

## Headers

This route requires the `Authorization` header with the JWT.

```
Authorization: Bearer (token)
```

## Route Parameter

This route requires the folder path to be specified in the URL.

Some examples include:

-   `http://localhost:3000/api/files/list/`: Lists the contents of the root
    directory of the current user
-   `http://localhost:3000/api/files/list/abc/`: Lists the contents of the
    abc folder.

## Response

This route can return HTTP code 200 (OK), 401 (Unauthorized), 404 (Not Found),
405 (Method Not Allowed), and 500 (Internal Server Error)
